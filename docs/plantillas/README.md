# Plantillas

## Propósito
Almacena plantillas reutilizables para documentación técnica, funcional y operativa de los proyectos.

## Resumen del directorio
El uso de plantillas estandariza la forma de documentar requerimientos, manuales, reportes y decisiones, reduciendo variaciones entre equipos.

## Ejemplos de qué puede escribirse aquí
- Plantilla de requerimiento funcional
- Plantilla de manual de usuario
- Plantilla de documento técnico
- Plantilla de reporte de pruebas
