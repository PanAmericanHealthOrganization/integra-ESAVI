# Diagramas de Arquitectura

## Propósito
Contiene diagramas que representan visualmente la estructura, comportamiento e integración de la plataforma.

## Resumen del directorio
Aquí se almacenan diagramas actualizados para facilitar la comunicación técnica entre equipos funcionales y de desarrollo.

## Ejemplos de qué puede escribirse aquí
- Diagrama de contexto (C4 nivel 1)
- Diagrama de contenedores (C4 nivel 2)
- Diagrama de flujo ETL entre fuentes y base de integración
- Diagrama de despliegue en ambientes
