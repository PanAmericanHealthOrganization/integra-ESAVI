# Integraciones

## Propósito
Documenta cómo la plataforma se integra con sistemas externos y fuentes de datos institucionales.

## Resumen del directorio
Esta carpeta define contratos de intercambio, mapeos de campos, periodicidad de sincronización y manejo de errores por integración.

## Ejemplos de qué puede escribirse aquí
- Especificación de integración con DHIS2 Tracker
- Especificación de integración con VigiFlow
- Mapeo de campos de la base de vacunación al esquema unificado
- Políticas de reintento y recuperación de fallas
