# Arquitectura

## Propósito
Agrupa la documentación de arquitectura de la plataforma ESAVI: visión técnica, decisiones de diseño, diagramas y descripción de integraciones entre sistemas.

## Estructura
```
arquitectura/
├── diagramas/      # Diagramas de alto y bajo nivel (C4, flujos, despliegue)
├── documentos/     # Documentos técnicos y decisiones de arquitectura
├── integraciones/  # Especificaciones de integración entre sistemas
└── resumen.md      # Resumen ejecutivo de arquitectura
```

## Resumen del directorio
Esta carpeta ayuda a entender cómo se conectan los componentes de la plataforma y por qué se tomaron ciertas decisiones técnicas.

## Ejemplos de qué puede escribirse aquí
- Arquitectura lógica de la plataforma por capas
- Decisión de diseño para el middleware de integración
- Flujo de datos entre DHIS2, VigiFlow y vacunación
- Matriz de dependencias entre sistemas
