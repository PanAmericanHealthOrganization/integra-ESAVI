# Glosario

## Propósito
Define términos técnicos, funcionales y del dominio de vigilancia para un lenguaje común en todo el equipo.

## Resumen del directorio
Este directorio evita ambiguedades en la interpretación de documentos, metadatos y reportes del proyecto.

## Ejemplos de qué puede escribirse aquí
- Definiciones de ESAVI y EVADIE
- Términos DHIS2 (Tracked Entity, Program, Event)
- Definiciones de indicadores analíticos
- Siglas institucionales y operativas
