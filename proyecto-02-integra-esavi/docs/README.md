# Documentación — Integra-ESAVI

## Propósito
Consolida la documentación funcional, técnica y operativa del proyecto de integración Integra-ESAVI.

## Estructura
```
docs/
├── API/            # Especificaciones de endpoints, contratos y ejemplos
├── database/       # Modelo de datos, migraciones y reglas de persistencia
├── capacitacion/   # Materiales de formación para equipos técnicos y funcionales
├── gestion-cambio/ # Planes, actas y evidencias de adopción
├── manuales/       # Manuales por perfil (usuario, técnico, desarrollador)
└── requerimientos/ # Requerimientos funcionales y no funcionales
```

## Resumen del directorio
Esta carpeta es la referencia principal para implementar, operar y evolucionar el middleware de integración.
