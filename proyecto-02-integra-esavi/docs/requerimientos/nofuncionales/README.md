# Requerimientos No Funcionales

## Propósito
Especifica atributos de calidad y restricciones técnicas del sistema.

## Ejemplos de qué puede escribirse aquí
- Objetivos de disponibilidad y tiempos de respuesta
- Requisitos de seguridad y control de acceso
- Requisitos de auditoría y trazabilidad
- Requisitos de mantenibilidad y observabilidad
