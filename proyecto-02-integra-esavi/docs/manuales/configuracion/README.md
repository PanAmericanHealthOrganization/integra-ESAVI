# Manual de Configuración

## Propósito
Describe cómo configurar ambientes, variables y dependencias para ejecutar Integra-ESAVI.

## Ejemplos de qué puede escribirse aquí
- Configuración de variables de entorno
- Parametrización de conexiones a DHIS2 y VigiFlow
- Configuración de base de datos y puertos
- Validación inicial post-despliegue
