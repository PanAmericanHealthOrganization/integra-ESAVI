# Contenedores

## Propósito
Contiene definiciones de contenedores y orquestación para desplegar los servicios de Integra-ESAVI.

## Ejemplos de qué puede escribirse aquí
- Archivos Dockerfile por servicio
- Docker Compose para entorno local
- Variables y secretos requeridos por contenedor
- Notas de despliegue por ambiente
